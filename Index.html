Transformations of Functions Study Guide

1. Vertical Shifting

Vertical shifting moves the graph up or down without changing its shape.

y = f(x) + k
Transformation Equation Effect
Shift Up y = f(x) + k (k > 0) Graph moves up by k units
Shift Down y = f(x) - k (k > 0) Graph moves down by k units
Example: If f(x) = x², then f(x) + 3 = x² + 3 shifts the parabola up by 3 units.

2. Horizontal Shifting

Horizontal shifting moves the graph left or right without changing its shape.

y = f(x - h)
Transformation Equation Effect
Shift Right y = f(x - h) (h > 0) Graph moves right by h units
Shift Left y = f(x + h) (h > 0) Graph moves left by h units
Example: If f(x) = x², then f(x - 2) = (x - 2)² shifts the parabola right by 2 units.

3. Reflecting Graphs

Reflections create mirror images of the graph across axes.

y = -f(x) or y = f(-x)
Transformation Equation Effect
Reflection over x-axis y = -f(x) Graph flips vertically
Reflection over y-axis y = f(-x) Graph flips horizontally
Example: If f(x) = x², then -f(x) = -x² reflects the parabola over the x-axis.

4. Vertical Stretching and Shrinking

Vertical stretching/shrinking changes the height of the graph.

y = a·f(x)
Transformation Equation Effect
Vertical Stretch y = a·f(x) (|a| > 1) Graph becomes taller
Vertical Shrink y = a·f(x) (0 < |a| < 1) Graph becomes shorter
Example: If f(x) = x², then 2f(x) = 2x² vertically stretches the parabola by a factor of 2.

5. Horizontal Stretching and Shrinking

Horizontal stretching/shrinking changes the width of the graph.

y = f(b·x)
Transformation Equation Effect
Horizontal Shrink y = f(b·x) (|b| > 1) Graph becomes narrower
Horizontal Stretch y = f(b·x) (0 < |b| < 1) Graph becomes wider
Example: If f(x) = x², then f(2x) = (2x)² horizontally shrinks the parabola by a factor of 2.

6. Even and Odd Functions

Even and odd functions have specific symmetry properties.

Type Definition Symmetry Example
Even Function f(-x) = f(x) Symmetric about y-axis f(x) = x², f(x) = cos(x)
Odd Function f(-x) = -f(x) Symmetric about origin f(x) = x³, f(x) = sin(x)
Neither Doesn't satisfy either condition No specific symmetry f(x) = x² + x
Test for Even/Odd:
1. Replace x with -x in the function
2. Simplify the expression
3. Compare with original function and -f(x)
4. If f(-x) = f(x), it's even; if f(-x) = -f(x), it's odd; otherwise, neither.
